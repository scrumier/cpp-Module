# cpp-Module

A series of C++ projects covering the fundamentals of object-oriented programming.

## What it does

This repository contains modules 00 through 09 from the School 42 C++ curriculum. Each module introduces new concepts through small exercises and projects.

## Modules overview

| Module | Topics |
|--------|--------|
| 00 | Namespaces, classes, member functions, I/O streams |
| 01 | Memory allocation, pointers to members, references, file streams |
| 02 | Ad-hoc polymorphism, operator overloading, fixed-point numbers |
| 03 | Inheritance |
| 04 | Subtype polymorphism, abstract classes, interfaces |
| 05 | Repetition and exceptions |
| 06 | C++ casts |
| 07 | Templates |
| 08 | Templated containers, iterators, algorithms |
| 09 | STL containers applied to a practical problem |

## Structure

Each module lives in its own directory (`cpp00` through `cpp09`) and contains one or more exercises, each in a separate subdirectory.

```sh
cd cpp00/ex00
make
```

## Project context

This is the C++ track at School 42. The goal is to learn C++ from scratch, with a focus on object-oriented design and the features that distinguish C++ from C.
